#🎄 X-masGame 🎮

Unity로 제작한 간단한 크리스마스 테마 점프 액션 게임

---

## ✨ 주요 기능

- 플레이어 이동 & 점프
- 크리스마스 테마 스테이지
- 적 캐릭터와 아이템 획득
- 스테이지 이동 및 엔딩 화면

## ▶️ 실행 방법

1. Unity Hub에서 프로젝트 열기  
2. Unity 2021.3 LTS 이상 권장  
3. `Assets/Scenes/GameStart` → Play 버튼 클릭!
